> 设计日期：2026-02-03

## 📋 概述

## 🎯 核心目标

创建一个**通用的、编辑器无关的、AI 驱动的自动化系统**：

1. **快速理解** - AI 在新对话中能快速理解项目结构
2. **自动维护** - 基于 Git Hook 自动触发文档更新
3. **增量更新** - 智能判断变更影响，只更新必要内容
4. **编辑器无关** - 适用于任何 IDE 和 AI 助手
